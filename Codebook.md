## Study design

In order to determine the good names to use for the columns, I've listed all original names of activities. Then I've defined the bes names to use to get more readable names:

* -mean() => Mean
* -std() => STD
* -freq() => Frequency
* Acc => Accelerometer
* Gyro => Gyroscope
* BodyBody => Body
* Mag => Magnitude
* ^t => Time
* ^f => Frequency
* tBody => TimeBody
* angle => Angle
* gravity => Gravity
